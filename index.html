PORTFOLIO

Confira meus projetos

Spotify

Este projeto é um clone da interface do Spotify, desenvolvido com o objetivo de praticar e demonstrar habilidades em HTML, CSS e responsividade. A proposta é replicar visualmente a plataforma de streaming, trazendo elementos como a sidebar de navegação, capa de álbuns, player de músicas e organização das seções — tudo com uma aparência moderna e parecida com o serviço original. É uma ótima forma de estudar posicionamento de elementos, estilos personalizados e construção de interfaces musicais atrativas.

Finans

O Finans é um aplicativo criado para ajudar você a organizar suas finanças pessoais de forma simples, rápida e inteligente. Com ele, é possível controlar seus gastos, planejar seu orçamento, economizar mais e alcançar seus objetivos financeiros. Tudo isso com uma interface amigável, acessível de qualquer lugar e com recursos pensados para facilitar o seu dia a dia.

Acesse

Despesas

O site de contar despesas é uma aplicação voltada para o controle financeiro pessoal. Com uma interface simples e intuitiva, o usuário pode registrar seus gastos e receitas, tendo uma visão clara de quanto está ganhando, gastando e economizando. A proposta é facilitar o dia a dia de quem quer organizar suas finanças sem complicação, com recursos essenciais como cadastro de transações, categorias e saldo atualizado automaticamente. Ideal para quem busca praticidade na hora de cuidar do dinheiro!
